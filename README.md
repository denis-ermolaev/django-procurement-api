# 1. About ----

**REST API сервиса закупок** на Django REST Framework. Покупатель работает с каталогом, корзиной, адресами и заказами; магазин управляет своими предложениями и позициями заказов; администратор модерирует магазины, каталог, предложения, пользователей и заказы. Проект реализован как итоговый по курсу Нетологии по веб-разработке. Он **разрабатывается в формате AI-assisted development**
